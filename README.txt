BLOODBERSS — сборка в .exe
============================

Что нужно: установленный Node.js (скачать на nodejs.org, если ещё нет — версия LTS подойдёт).

Шаги:

1. Распакуй этот архив в любую папку.
2. Открой в этой папке терминал / PowerShell / cmd.
3. Выполни по очереди:

   npm install
   npm run dist

4. Подожди (первая сборка качает Electron, может занять пару минут).
5. Готовый установщик появится в папке dist\ — файл вида
   "BLOODBERSS Setup 1.0.0.exe"

Запусти его — поставится как обычная программа, с ярлыком на рабочем столе и
своей иконкой (котик-волна).

Лайки/коллекция сохраняются локально на твоём ПК (через localStorage Electron-приложения), между запусками не теряются.

Если захочешь поменять иконку — положи свой .ico файл в build/icon.ico
(заменив существующий) и пересобери (npm run dist).

------------------------------------------------------------
Что изменилось в этой версии
------------------------------------------------------------
- Все SVG-иконки в пустых состояниях, лоадерах и кнопке лайка заменены
  на символьных котиков (эмодзи).
- Убран визуальный эквалайзер в плеер-баре.
- На вкладке "Волна" появились кнопки типов волны: Любая / Русское /
  English / Funk / Спокойные / Грустные / Весёлые. Клик переключает
  фильтр и сразу перезагружает подборку.
- Поиск (и "Русская" волна) теперь подмешивает дополнительные запросы
  для кириллических названий/исполнителей и поднимает такие треки
  выше в списке — это не даёт гарантии "только русское", т.к. у Audius
  нет официального фильтра по языку трека, но заметно увеличивает долю
  русскоязычных результатов.
- Новая иконка приложения (build/icon.ico, build/icon.png) и небольшое
  лого-лицо кота в шапке плеера.

Что поменялось в этой версии
=============================

1. Сервер зашит по умолчанию: http://192.168.0.143:4000
   Экран "укажи сервер" больше не всплывает при обычном запуске —
   сразу идёт на экран входа. Поменять адрес всё ещё можно вручную:
   кнопка "Сменить адрес сервера" в настройках/профиле (на случай,
   если роутер выдаст телефону другой IP).

2. Ограничение без BLOODPLUS: трек теперь играет первые 10 секунд,
   потом ставится на паузу и всплывает баннер с приглашением оформить
   подписку через Telegram (@bloodberss). Кнопка play/pause тоже не
   даёт продолжить воспроизведение после лимита. Админы и пользователи
   с активным BLOODPLUS (user.plus.active = true на сервере) слушают
   без ограничений.

3. На экране профиля у тех, у кого нет BLOODPLUS, теперь карточка с
   приглашением в Telegram (кликабельная ссылка @bloodberss).

4. Внешние ссылки (Telegram и подобные) теперь открываются в обычном
   браузере системы, а не внутри окна приложения — поправлено в main.js.

5. Починена кнопка "Выдать BLOODPLUS" в админке — раньше она не открывала
   форму выдачи из-за опечатки в названии формы (plus vs plus-grant).

6. Полоса прогресса трека и громкости теперь заливаются красным по мере
   движения (а не просто двигающаяся точка на сером фоне).

7. В профиле появилась загрузка своей аватарки (кружок с карандашиком
   поверх аватара → выбираешь картинку → она автоматически уменьшается
   и сохраняется на сервере). Требует обновлённого server.js
   (добавлен эндпоинт POST /api/user/avatar).

Сборка — как раньше:
   npm install
   npm run dist
